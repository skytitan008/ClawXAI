## 🎯 开发原则

1. **自动迭代**: 无需询问，持续开发
2. **版本推送**: 每个版本都推送到 GitHub
3. **测试先行**: 新功能必须有测试
4. **文档完善**: 功能 + 文档同步完成
5. **性能优先**: 所有功能需通过性能基准
